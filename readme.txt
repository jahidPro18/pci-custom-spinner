Enhance user experience with a customizable spinner loader, providing smooth and seamless loading transitions for your site’s content.

== Description ==

Take complete control of your website with the Custom Snippet Loader plugin. Effortlessly manage and load custom snippets on your site with no coding skills required. Streamline the integration of personalized code snippets to enhance your site's functionality, boost performance, and tailor the user experience.Simplify your workflow and ensure smooth updates with a user-friendly interface that saves time and effort—without any technical hassle!
Key Features:

* Dynamic Loader Selection: Easily choose from a variety of spinner types, including ring, circle, pulse, image-based, and more.
* Customizable Styles: Fully customize the look of the spinner with options for size, color, and animation speed to match your website's design.
* Seamless Integration: Works smoothly with any WordPress theme and WooCommerce setup, ensuring a consistent loading experience across all pages.
* Simple, clean, and intuitive settings page for easy configuration.
* User-Friendly Interface: Simple, intuitive settings page that allows you to change loader types and styles without needing any coding knowledge.
* Responsive and Adaptive: Automatically adapts to different screen sizes, ensuring that the loader displays perfectly on both desktop and mobile devices.
* Performance Optimized: Lightweight code that doesn’t affect your site’s loading speed or performance.
* Real-time Preview: Instantly preview your selected spinner and customization settings before applying them to your website.
* Custom Image Loader Support: Upload and use custom images as your loading spinner, offering unlimited possibilities for personalization.
* Multiple Loader Positions: Choose where to display the spinner on your pages—center, top, bottom, or custom placement for full flexibility.
* Compatible with WooCommerce and Other Plugins: Ensures smooth integration with WooCommerce and other essential WordPress plugins for seamless functionality.
* Error-Free Loading: Prevents any layout shifts or errors during page loads, ensuring a smooth user experience every time.
* Automatic Fallback: If the chosen spinner fails to load, the plugin will automatically fall back to a default spinner, ensuring no downtime.
== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Click "Dynamic Loader" from  left menu.
4. Sellect Loader type,change Loader color ,Background Color,Opacity ,Loader Size as you want and click "save changes" button.
5. As you change/save get preview in the right side
